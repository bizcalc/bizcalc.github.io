BizCalc Privacy Policy

What is BizCalc

BizCalc is a simple application that help user to calculate the projection of a business proposal. It help to make calculation based on some input data that user provide and then turn it into business analysis metric that can help the user to make a fast decision whether a project worth pursuing.

Privacy Matter

BizCalc doesn't collect and store any of user data in any way. So user privacy won't be violated in in any way when they are using this application.
